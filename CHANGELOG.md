# 版本说明

## 1.1.0

- 支持在单个窗口中同时打开多个 Markdown 文件，并可在左侧栏切换、关闭和拖拽排序
- 支持将应用注册为 Markdown 文件默认打开方式，双击或从系统打开文件会复用现有窗口
- 侧边栏支持展开和收起，文件较多时文档列表可滚动浏览
- 支持刷新当前 Markdown 文件，便于同步外部编辑器对文件的修改
- 移除顶部重复的新建和打开入口，统一保留左侧栏操作入口
- 支持拖动编辑区与预览区之间的分割条调整显示比例
- 修复分割条松开后仍跟随指针移动的问题
- 优化展开侧边栏时的应用图标显示
- 支持 Markdown 任务复选框在预览区直接勾选和取消勾选
- 修复 Mermaid 图表首次渲染时横向图表被缩得过小的问题

## 1.0.0

- 将应用统一为 `alun reader`
- 支持 macOS、Windows、Linux Electron 桌面端打包
- 增加真实文件保存、另存为、导入和最近打开列表
- 支持多窗口新建和在新窗口中打开文件
- 优化 Markdown 表格、图片、行内代码和深色模式渲染
- 优化 Mermaid 主题、节点文字对比、缩放、重置和全屏查看
- 增加 Zen 模式，便于投屏讲解
- 移除无效的假入口，减少误导性 UI
- 增加 PolyForm Noncommercial License 1.0.0，明确禁止商用，并允许所有开源非商用项目使用
- 增加 GitHub Pages 发布页和基于 GitHub Release 的更新检查
- 增加 Release 发布脚本，Release 不存在时可创建并上传关键构建产物
- 应用内更新检查支持版本比较、平台安装包匹配、下载并打开安装包
- 更新检查增加 GitHub Pages 静态清单兜底，避免 GitHub API 匿名限流导致 403 后完全不可用
